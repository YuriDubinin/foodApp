## Setup

```bash
$ git clone https://github.com/YuriyDubinin/foodApp.git
$ cd foodApp
$ npm i
$ npx json-server db.json
$ npx webpack
```

## Build

```bash
It is required to set the "production" mode in the settings webpack.config.js
```
